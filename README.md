🇬🇧 AI fact-checking Telegram bot — pulls text from a shared link or claim, runs it through an LLM via OpenRouter (Qwen / Nemotron), and replies with a verdict, confidence score, and short analysis. History is logged to JSON.<br/>
🇷🇺 Telegram-бот для AI-факт-чекинга: забирает текст по ссылке или из сообщения, прогоняет через LLM через OpenRouter (Qwen / Nemotron) и присылает вердикт, оценку уверенности и краткий разбор. История хранится в JSON.
